### <a id="whoami"></a>`whoami`

I'm **Stella**, a developer who wrote her first lines of code at 11, built her own PC from parts at 13, and that curiosity has only compounded since.

These days I'm most at home in **cybersecurity, systems, and infrastructure**: I run my own self-hosted services, care a lot about how networks actually work, and do all of it on **Debian** (under Proxmox if in a server environment), which has also been my daily driver for years in the form of **LMDE** (Linux Mint Debian Edition).

I also enjoy disassembling servers, routers and other hardware to fix or sometimes jailbreak it, and I have an interest in **reverse engineering** and **embedded systems**, which I explore through my own projects and IOS security research.

### <a id="homelab"></a>`homelab`

I run a small fleet of self-hosted services on my own hardware, and it's where most of my sysadmin, networking, and security learning actually happens. 

```text
~/infra
├── linux servers ......... debian, proxmox
├── self-hosted services .. pi-hole, openclaw, vaultwarden, openwebui
├── networking ............ tailscale, localsend, pfsense
└── security .............. crowdsec, honeypots, custom scripts
```
